# Humankind Mod

A mod project for [Humankind](https://www.humankindthegame.com/) (Amplitude
Studios / SEGA). **No specific mod concept has been chosen yet** — this repo
is currently just the AI-collaboration scaffold: a meta-prompt, ground-truth
research, and reusable research-brief templates, carried over from the
methodology developed on
[civ7-netaji-mod](https://github.com/ovhirup/civ7-netaji-mod).

## Layout

```
CLAUDE.md               Working instructions + methodology (read every session)
GROUND-TRUTH.md          Detailed, cited research on how Humankind modding actually works
DESIGN.md                Roadmap — empty until a mod concept is picked
grok-project/            Reusable Advisor (external research) prompt templates
```

## Status

Scaffold only. Next step: install "Humankind Mod Tools" from Steam Library →
Tools and verify the claims in GROUND-TRUTH.md hands-on, then pick a mod
concept and fill in DESIGN.md.
